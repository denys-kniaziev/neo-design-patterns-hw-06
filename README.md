# Woolf University. Design Patterns Course. Homework – Decorator and Proxy Patterns

Practice structural design patterns in TypeScript.

## Topic

Build a message service using:

* Decorator
* Proxy
* Interface-based design
* Rate limiting

## Main Logic

The app should demonstrate:

* Sending messages through a common interface
* Modifying messages with decorators
* Adding timestamps
* Converting messages to uppercase
* Limiting message frequency with a proxy
* Blocking requests sent too frequently
* Extending behavior without changing the base service

## Run

```bash
npx ts-node src/main.ts
```
